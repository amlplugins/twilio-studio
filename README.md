# @amlplugins/twilio-studio

Native `twilio` SDK re-exported under the `@amlplugins` namespace.

> Twilio Studio — visual workflow IDE for orchestrating multichannel customer journeys.

## Install

```bash
npm install @amlplugins/twilio-studio
```

## Use

```ts
import { /* named exports from twilio */ } from "@amlplugins/twilio-studio";
// or
import sdk from "@amlplugins/twilio-studio";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
